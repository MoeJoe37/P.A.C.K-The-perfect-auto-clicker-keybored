# PACK 🎯
**Automation macro tool — record, sequence, and execute actions effortlessly.**

---

## What is PACK?

PACK is a desktop automation utility that lets you record mouse clicks and keyboard inputs, organize them into reusable sequences, and replay them on demand — all controlled via customizable hotkeys.

---

## Features

### 🎬 Action Recording
- Hit the **Record** button to minimize PACK and start capturing your actions in the background.
- Recording runs freely until you press your configured **hotkey** to stop — so there's no time pressure.
- Each recorded action appears in the action list. Mouse clicks automatically get a **"Hold"** checkbox, letting you hold that button down instead of a single click.
- Once done, save the recording directly to a **file location** or drop it straight into the **Sequences tab**.
- Enable **Repeat Forever** on any action to loop it continuously until the hotkey is pressed again.

> ⚠️ The Record button won't function until a hotkey is set — go to Settings first.

---

### 📋 Sequences Tab
Build multi-step automation flows by combining recorded actions into sequences.

| Button | Description |
|---|---|
| ▶ **Run** | Execute the sequence immediately |
| 💾 **Save Sequence** | Save the sequence to disk for later use |
| 📤 **Export** | Export the sequence to a shareable file format |

- After a sequence finishes executing, PACK automatically **returns to full screen**.
- Pressing the hotkey while a sequence is running will **minimize PACK** at any time.

---

### ⌨️ Hotkey Support
- Set a global hotkey in Settings.
- The hotkey controls everything: start/stop recording, minimize/restore the window, and interrupt running sequences.
- PACK must have a hotkey configured before recording is available.

---

## Getting Started

1. **Set your hotkey** — open Settings and bind a key combination of your choice.
2. **Record an action** — click **Record** in the main menu. PACK will minimize; perform your actions, then press the hotkey to stop.
3. **Review your recording** — mouse click actions will show a **Hold** checkbox. Enable it if you want the click held down.
4. **Save or sequence it** — save the recording to a file or add it to a sequence in the Sequences tab.
5. **Run it** — head to the Sequences tab and hit **Run**.

---

## Requirements

- Windows 10 / 11
- No additional dependencies required

---

## Tips

- Use **Repeat Forever** for actions you want looped — like holding a key or spamming a click — and stop them instantly with the hotkey.
- Build complex automations by chaining multiple saved recordings into a single sequence.
- Export sequences to share them with others or back them up.
